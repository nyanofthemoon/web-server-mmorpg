Applet Configuration


[Home | Administrator Guide]

You can modify this page to allow the VolanoChat server administrator to modify the applet properties from any Web browser. You first need to make the following changes:

  1. Change the links listed below in the source of this Web page from localhost:8080 to the actual host name and port number of the VolanoChat servlet runner on your system. The servlet runner uses port 8080 by default.
  2. Define an administrator password by setting the VolanoChat server property admin.password in the server properties file (called properties.txt by default). You will be prompted for a name and password by the applet configuration servlet after you click on the modified links below. When prompted, you can use any name, but the password must match the server's admin.password value.
  3. Change the VolanoChat server property applet.codebase to point to the full path of your VolanoChat applet code base directory. You must specify the local path name on the server rather than the external Web page address.

After making those changes, simply click a link below to modify the corresponding applet properties:
