Hidden Personal Room

This is the MyVolanoChat applet. It opens directly into a personal room which is associated with only this Web page and is hidden from view in the public VolanoChat server.

Public Personal Room

This is also the MyVolanoChat applet. It opens directly into a personal room but is also added to the list of public rooms available in the VolanoChat server.

The applet tag for this applet is the same as the first one except for the following additional parameter:

<param name="public" value="true" />
