Embedded Public Rooms

This is the WebVolanoChat applet. It remains embedded on the Web page yet still allows multiple independent private chat rooms among its visitors if permitted by the server administrator.

Automatic Entry

The visitor will enter the room automatically if a name is provided as an applet parameter, as shown below:

The applet tag for this applet is the same as the first one except for the following additional parameters:

<param name="username" value="John" />
<param name="profile"  value="http://www.volano.com/" />
