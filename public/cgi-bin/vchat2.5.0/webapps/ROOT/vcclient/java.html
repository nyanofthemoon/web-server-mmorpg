Unsupported Java Version

Problem

The VolanoChat applet failed to connect to the VolanoChat server because the server requires functions from the applet that are not available with the older version of Java you are using.

Solution

You can access the VolanoChat server by upgrading your Web browser to a version that provides the required Java support. Visit the Web Standards Project Browser Upgrades page for the download locations of Web browsers that support accepted Web standards. Make sure to select Java support for the browser if it's provided as an optional package during the installation.

Windows 95 Notes

If you are running Windows 95, you should upgrade to either Internet Explorer 5.5 or Netscape 6, since Internet Explorer 6.0 is not available for Windows 95. Note that Internet Explorer 5.5 will run on older, less powerful computers, while Netscape 6 requires more system resources. See the system requirements of each browser for details.

If that didn't work ...

If you still don't have the required Java support in your Web browser after upgrading to one that supports Web standards, you can get the latest Java Plug-in for your browser directly from the Sun Get Java page. Sun's Java Plug-in provides the required support for connecting to this VolanoChat server.

For a detailed list of the Web browsers which work with VolanoChat, see the VolanoChat Browser Support page.
